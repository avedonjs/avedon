# Support

## Where to get help

| Topic | Channel |
|-------|---------|
| Bug reports | [GitHub Issues](https://github.com/hocestnonsatis/vexjs/issues) — use the Bug report template |
| Feature requests | [GitHub Issues](https://github.com/hocestnonsatis/vexjs/issues) — use the Feature request template |
| Usage / how-to questions | [GitHub Discussions](https://github.com/hocestnonsatis/vexjs/discussions) if enabled; otherwise open an Issue |
| Security vulnerabilities | See [SECURITY.md](./SECURITY.md) — do not use public issues |

## Documentation

- [README](./README.md) — overview and quick start
- [`docs/`](./docs/) — design notes and deeper documentation

Before opening an issue, search existing issues and skim the README for known behavior.
