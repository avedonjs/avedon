# Support

## Where to get help

| Topic | Channel |
|-------|---------|
| Bug reports | [GitHub Issues](https://github.com/avedonjs/avedon/issues) — use the Bug report template |
| Feature requests | [GitHub Issues](https://github.com/avedonjs/avedon/issues) — use the Feature request template |
| Usage / how-to questions | [GitHub Discussions](https://github.com/avedonjs/avedon/discussions) if enabled; otherwise open an Issue |
| Security vulnerabilities | See [SECURITY.md](./SECURITY.md) — do not use public issues |

## Documentation

- [README](./README.md) — overview and quick start
- [Documentation index](./docs/README.md) — guides for components, routing, rendering, and packages

Before opening an issue, search existing issues and skim the README for known behavior.
