# Docs Site (`apps/www`) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship `apps/www` — avedon dogfood landing + docs from `docs/*.md` (build-time markdown → HTML), SSG-only, deployable to Cloudflare Pages via `build/client`.

**Architecture:** Add `{@html}` so trusted markdown HTML can render. A generate script writes `apps/www/.generated/docs.json` from the include list. `Home.ave`, `DocsIndex.ave`, and `Doc.ave` are all `render: 'ssg'`; `adapter-node` still runs at build time to emit static HTML; only `build/client` is published to Cloudflare Pages.

**Tech Stack:** avedon / `@avedon/*` (workspace), `marked`, Node `fs`, Vitest for the generate script, Cloudflare Pages (static)

**Spec:** `docs/superpowers/specs/2026-07-22-docs-site-design.md`

## Global Constraints

- Stay on `main`; do not create feature branches
- Commit only when the maintainer explicitly asks (include `git add`/`commit` steps below but skip them until asked)
- TypeScript 5.x only (no TS 6 bump)
- English-only docs and UI copy
- Wordmark: lowercase `avedon`
- Palette: `--bg #09090B`, `--fg #FAFAFA`, `--muted #A1A1AA`, `--accent #06B6D4`, `--accent-deep #0891B2`, `--line rgba(250,250,250,0.12)`
- Font: Syne; no Inter/Roboto/Arial/system as primary
- No Tailwind / CSS frameworks
- Landing: no feature grids, stat strips, card heroes, or floating badges
- v1 docs include only: `guide`, `avedon-components`, `routing`, `rendering`, `middleware`, `session`, `security`, `packages`
- Exclude: `docs/README.md`, `docs/publishing.md`, `docs/superpowers/**`
- No search, versioning, i18n, Workers SSR, or www Playwright in this plan
- Root `pnpm build` (turbo) must not require building `www`; site builds via `pnpm -F www build`

```ts
import fs from 'node:fs'
import path from 'node:path'
import { fileURLToPath } from 'node:url'

export type DocHeading = { id: string; text: string; level: 2 | 3 }
export type DocEntry = {
  slug: string
  title: string
  blurb: string
  html: string
  headings: DocHeading[]
}
export type DocsFile = { docs: DocEntry[] }

const generated = path.resolve(
  path.dirname(fileURLToPath(import.meta.url)),
  '../../.generated/docs.json',
)

export function loadDocsFile(): DocsFile {
  if (!fs.existsSync(generated)) {
    throw new Error('Missing .generated/docs.json — run pnpm -F www generate')
  }
  return JSON.parse(fs.readFileSync(generated, 'utf8')) as DocsFile
}

export function listDocs(): DocEntry[] {
  return loadDocsFile().docs
}

export function getDoc(slug: string): DocEntry | undefined {
  return listDocs().find((d) => d.slug === slug)
}

export function docStaticPaths(): string[] {
  return listDocs().map((d) => `/docs/${d.slug}`)
}
```

```ave
  export let html
  export let headings
  export let docs
  export let slug
</script>

<script server>
  import { notFound } from '@avedon/server'
  import { getDoc, listDocs } from '../lib/docs'

  export async function load({ params }) {
    const entry = getDoc(params.slug)
    if (!entry) throw notFound()
    return {
      title: entry.title,
      html: entry.html,
      headings: entry.headings,
      docs: listDocs().map((d) => ({ slug: d.slug, title: d.title })),
      slug: entry.slug,
    }
  }
</script>
```
